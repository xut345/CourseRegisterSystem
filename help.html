Tutorial

View available courses and register a course:

1. Click "Courses" in the top menu.

2. Click "term" select box to select a term.

3. Select the program at the yellow tab, you can only select one program once.

4. Click the black "Submit" button.

5. All courses are showing under the submit button now. If needed, you can also translate the current page to other languages by clicking the google translate box at left corner.

6. If you want to find a particular course, entre the course ID in the "Course ID" search box. If you want to go to the top of current page, click the red "top" button at right corner.

7. Click "view sections" to view the course' details.

8. The table will show all sections in the current course. Select one section that you want to regist. Then click the black "Register" button.


Drop a course:

1. Click "My Information" in the top menu.

2. Click "term" select box to select a term.

4. Click the black "Submit" button.

5. All courses you registed this term are showing under the submit button now.

8. Select the courses you want to drop. Then click the black "Drop" button to drop the courses.
